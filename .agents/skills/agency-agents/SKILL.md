## Usage

To use any agent, read the corresponding `.md` file for the full system prompt and instructions. These can be used to guide specialized analysis, delegated to subagents, or referenced for domain-specific best practices.

Example: To run a security review, read `engineering/engineering-security-engineer.md` and use its guidance for threat modeling and secure code review.
