## PART 2: UPDATE PRICING LOGIC

Remove ALL special fee calculations from compute_pricing.
Price is based ONLY on cubic yards and job type.
```python
def compute_pricing(items, cy_low, cy_mid, cy_high,
                    rate_low, rate_high, premium_rate,
                    min_charge, is_premium):
    
    if is_premium:
        price_low = cy_low * premium_rate
        price_high = cy_high * premium_rate
    else:
        price_low = cy_low * rate_low
        price_high = cy_high * rate_high
    
    # Apply minimum charge
    price_low = max(price_low, min_charge)
    price_high = max(price_high, min_charge)
    
    # Collect special items for display only - no fee added
    special_items = [
        item for item in items 
        if item.get('is_special')
    ]
    
    return {
        'price_low': round(price_low, 2),
        'price_high': round(price_high, 2),
        'special_items': special_items
    }
```
